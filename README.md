## // About

<p align="center">
  <img src="https://readme-typing-svg.demolab.com?font=JetBrains+Mono&weight=500&size=16&duration=2800&pause=900&color=FF6B00&center=true&vCenter=true&width=560&lines=%3E+what+this+is;%3E+why+it+exists" alt="About typing" />
</p>

An image studio web app for editing, organising, and exporting visual assets - everything I would otherwise reach for a desktop tool to do, but in the browser. Optional AI-assisted tagging and captioning.

<a id="features"></a>

## // What's inside

<p align="center">
  <img src="https://readme-typing-svg.demolab.com?font=JetBrains+Mono&weight=500&size=16&duration=2800&pause=900&color=FF6B00&center=true&vCenter=true&width=560&lines=%3E+features+%C2%B7+goals" alt="Features typing" />
</p>

- `▸` Browser-based editing - crop, colour adjust, filter, watermark
- `▸` Asset library with tags, search, and folder-style collections
- `▸` Export presets for common targets (web sizes, social, print)
- `▸` AI-assisted tagging / captioning as an optional layer
